# GS-WebDev-FrontEnd

Este repositório contém o front-end do projeto SkillBridge — uma interface React + Vite construída para demonstrar páginas públicas (Home, About, Contact) e um painel (Dashboard) com cards de usuários, modais e formulários. O projeto usa Tailwind CSS para estilos e tem suporte a tema claro/escuro (pref. do sistema + toggle manual).

**Resumo do projeto**
- Stack: React, Vite, Tailwind CSS, React Router
- Objetivo: criar um front-end responsivo para listar profissionais, exibir detalhes em modais e permitir contato/registro/login (UI apenas; persitência depende de integração backend).
- Funcionalidades principais:
	- Páginas: Home, About, Contact, Dashboard
	- Componente de alternância de tema (dark / light)
	- Modais de Login / Register(em desenvolvimento...)
	- Formulário de contato com validação (react-hook-form)
	- Cards de usuários e modal de detalhes

**Instalação (passo-a-passo, Windows / PowerShell)**
1. Pré-requisitos:
	 - Node.js (recomendo v16+ ou v18+)
	 - npm (vem com o Node)
2. Clonar o repositório (se ainda não tiver):
```powershell
git clone https://github.com/MGmanfre/GS-WebDev-FrontEnd.git
cd GS-WebDev-FrontEnd/future
```
3. Instalar dependências:
```powershell
npm install
```
4. Rodar em modo de desenvolvimento:
```powershell
npm run dev
```
5. Abrir no navegador: normalmente o Vite abre em `http://localhost:5173` (ou mostre a URL no terminal).
